This is a self-contained, complete code sample that demonstrates how to mint a
Freeport token.

It uses a hard-coded private key, contract address etc in order to make the code
easy to follow.

# Pre-requisites

    - Node and NPM

# Setting up

    npm install

# Running the code

    npm start
